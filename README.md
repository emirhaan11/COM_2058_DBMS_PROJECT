Dump dosyası sql tableleri içeriyo, indirip onu direkt mysql databasesine ekleyebiliyosun.

IntellJ ile aç bence projeyi diğer türlü sıkıntı çıkarır.

.env dosyası oluşturman gerekiyor direkt projenin aana dizininie oluşturabilirsin.

içine yazman gerekenler aşağıda:

DB_URL=jdbc:mysql://localhost:3306/job_portal;  //sendeki de böyle bişidir büyük ihtimalle

DB_USER=root;     //databasenin adını giriyosun buda root dur büyük ihtimalle

DB_PASSWORD=      // kendi mysql kurarken koyduğun şifreyi yaz
